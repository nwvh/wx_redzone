# WX RedZone
Simple RedZone script for FiveM that shows blips of other players in the same zone

# Installation

## Manual
- Git Clone this repo
- cd into the cloned folder
- Run the following commands
- ```sh
  # using pnpm
  pnpm i
  pnpm build

  # using npm
  npm i
  npm run build
  ```
## Quick
- Download the release
